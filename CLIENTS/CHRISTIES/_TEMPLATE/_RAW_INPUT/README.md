# _RAW_INPUT — dépose ici tout le brut de la nouvelle villa

Aucune contrainte de format ou de tri. Glisse simplement :

- **Le dossier de présentation** (PDF de la brochure agence, fiche descriptive, mandat) — Claude en extrait le texte.
- **Les photos** (.jpg, .png) — toutes ensemble, en vrac. Claude les trie en hero / jour-nuit / 9 espaces / 28 galerie / portrait agent / vue localisation.
- **Les vidéos** (.mp4, .mov) — hero, films de pièces, aérien. Claude les trie aussi.
- **Tes notes** (`brief.txt`, `notes.md`, voix-en-texte) — le ton, ce qui rend cette villa singulière, l'angle éditorial, les éléments à mettre en avant.
- **Coordonnées agent** si différentes de Sylvie Koch (Christie's) — un simple `agent.txt` suffit.

## Si tu veux gagner du temps

Tu peux pré-remplir un `villa.json` partiel et le déposer ici — Claude reprendra ce qu'il y trouve et ne te demandera que ce qui manque.

## Une fois que tu as déposé

Ouvre Claude Code dans le dossier parent (celui qui contient ce `_RAW_INPUT/`) et dis : **"fais le site"**.
